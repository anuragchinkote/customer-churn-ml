# Customer Churn Prediction

This project predicts whether a customer will leave a telecom service.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Steps
- Data loading and cleaning
- Feature encoding
- Model training
- Model evaluation

## Models Used
- Logistic Regression
- Random Forest

## Result
Logistic Regression performed better than Random Forest on this dataset.

## Future Work
- Handle class imbalance
- Add more customer behavior features
